# Todo List

Difícil existir um programador que não fez um todo_list (rsrs) e eu não sou diferente. Esse é um projeto que fiz para me familiarizar ainda mais com a linguagem e colocar em prática meus conhecimentos. 

Como o nome já diz, é uma lista de tarefas, onde vc pode escolher a prioridade colocando cores e assim organizar as mais importantes.
